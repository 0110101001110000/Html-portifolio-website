Meu Portifólio




Bem-vindo ao meu portfólio digital! Abaixo, você encontrará alguns dos projetos em que trabalhei, destacando minhas habilidades como desenvolvedor de software.



Projeto: Aplicativo de Gerenciamento de Tarefas

Descrição: Este projeto consistiu no desenvolvimento de um aplicativo de gerenciamento de tarefas para dispositivos móveis Android. O aplicativo permite aos usuários criar, visualizar e gerenciar suas tarefas de forma eficiente. Utilizei Java como linguagem principal, juntamente com o framework Android SDK. Meu papel neste projeto foi desenvolver a funcionalidade de adicionar, excluir e marcar tarefas como concluídas.

Projeto: Sistema de Gestão de Clientes

Descrição: Este projeto envolveu o desenvolvimento de um sistema de gestão de clientes para uma empresa de pequeno porte. O sistema foi construído utilizando Spring Boot para o backend e React para o frontend. Fui responsável pelo desenvolvimento do backend, implementando as APIs RESTful para operações CRUD (Create, Read, Update, Delete) de clientes e suas informações.

Projeto: Blog Pessoal

Descrição: Desenvolvi um blog pessoal para compartilhar minhas experiências, conhecimentos e insights sobre desenvolvimento de software e tecnologia. Utilizei o framework Angular para criar a estrutura do blog e HTML/CSS para estilização. Além disso, integrei uma API RESTful para gerenciar os posts do blog. Como parte do projeto, também criei um painel de administração para gerenciar os posts.
